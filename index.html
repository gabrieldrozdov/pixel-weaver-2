PIXEL WEAVER 2.0

Info

What is this?

Pixel Weaver 2.0 is a generative art toy and tool that exploits fundamental properties of the CSS programming language to create dynamic compositions.

By changing a CSS property (like rotation) and setting a transition time, the CSS property automatically interpolates from one state to the next.

Numerous CSS properties can be combined to create complex outcomes. In particular, this project focuses on CSS grid and background settings. The name “Pixel Weaver” comes from the pixelated aesthetic that emerges when you zoom in an image to an extreme degree.

Record

Disclaimer

This site does not feature built-in recording functionality. Instead, it is much easier to use your computer’s built-in recording features.

I tried to implement this functionality directly into Pixel Weaver, but it wasn’t worth it because the quality ends up worse.

Apple Macbook/Mac Instructions

Use this keyboard shortcut for taking screenshots or screen recordings:

> Command + Shift + 5

Microsoft Windows Instructions

Use this keyboard shortcut to take a full-screen screenshot:

> Windows Key + PrtScn

Use this keyboard shortcut to take a full-screen recording:

> Windows Key + Alt + R

For more precision, use the built-in “Snipping” Windows application.

iPhone/iPad Instructions

For iPhones without a Home button, use the following shortcut to take a screenshot:

> Volume Up + Power

For iPhones with a Home button, use the following shortcut to take a screenshot:

> Home + Power

Newer iPhones can take screen recordings by adding a shortcut to your Control Center. To do this, go to Settings > Control Center and add the Screen Recording shortcut. You can then trigger a screen recording via the Control Center.

Android Instructions

Android screenshot shortcuts vary phone-to-phone. Generally, this shortcut works the most often:

> Volume Down + Power

Newer Android phones can take screen recordings via a shortcut in the Device Controls pulldown menu.
